Creating a game

Basic concepts

Rooms/Objects: Room and objects are both the same thing as far as the system is concerned.
Objects have numeric attributes and binary properties.
Attributes are set with dropdown menus. Properties are set with check boxes.
Functions: Code that is called to handle specific user input.
Events: Code that is called at the end of every turn.

Creating a New Game

Launch Lantern and select New Notice you have two objects by default, Offscreen and Player.
You can add new rooms and objects with the button.

Adding verbs

To a new verb, go to the verb tab, type it into the text box and click add.

Verbs can have checks placed on them.

Creating functions

Functions are created under the functions tab. Sentences call functions, so a function (even if it's just a stub needs) to be created first).
Give the function a name, then save it. You'll see it in the list of functions.
